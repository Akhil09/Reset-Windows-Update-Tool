Reset Windows Update Tool

Platforms: Desktop.
Requirements: Windows 7, Windows 8, Windows 8.1 and Windows 10.
Primary language: EN-US.
Author: Manuel Gil.
License: MLP.

Description:

Reset Windows Update Tool is a tool developed for use as a support of system repair options. This tool reset the Windows Update Components. Also is able to delete temporary files, scan, detect and repair corruptions with the Windows System image, scan all protected system files and replace any corrupted files, change invalid values in the Windows Registry, reset Winsock settings and more.
