Reset Windows Update Components

Reset Windows Update Components it is the first and main option to correct problems in Windows Update.

However , this process requires some threads to avoid problems during the search of a solution.

Suitable method to restore components:

Before starting, remember to make a backup the registry.

Then select the option "Reset Windows Update Components". This option will stop Windows Update services, while cleaning the components and restore the initial settings. e.g.

After, Install the latest Windows Update Agent https://support.microsoft.com/en-us/kb/949104.
