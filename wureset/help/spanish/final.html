Notas finales

Esta herramienta se desarrolló principalmente para restablecer componentes de Windows Update. A través del tiempo, se han añadido nuevas funciones para mejorar la experiencia de usuario.

Este documento ha sido creado para describir la secuencia de procesos para el uso de esta herramienta para restablecer componentes de Windows Update.

Referencias:

¿Cómo se pueden restablecer los componentes de Windows Update?: https://support.microsoft.com/es-us/kb/971058.

Use la herramienta Comprobador de archivos de sistema para reparar los archivos de sistema que faltan o están dañados: https://support.microsoft.com/es-us/kb/929833.

Corregir errores de Windows Update con las herramientas DISM o System Update Readiness: https://support.microsoft.com/es-us/kb/947821.

Glosario:

DISM: La administración y mantenimiento de imágenes de implementación es una herramienta de línea de comandos que se usa para realizar el mantenimiento de una imagen de Windows.

Fix It: programa de diagnóstico que detecta problemas de Windows.

Registro: Es una base de datos que almacena la configuración del sistema operativo.

Punto de restauración: es una representación de un estado almacenado de los archivos de sistema de su computadora.

SFC: System File Checker es una utilidad que permite a los usuarios buscar la corrupciones del sistema.

Winsock: Windows Sockets API es una especificación técnica que define cómo el software de red de Windows debe acceder a la red.
