Primeros pasos

La herramienta para restablecer los componentes de Windows Update tiene varias opciones de soporte como borrar los archivos temporales, cambiar los valores invalidos en el registro de Windows, restablecer las configuaciones Winsock, etc. Estas opciones aparecen en una lista de selección y están disponibles dependiendo del sistema operativo.

Funciones:

Esta herramienta implementa funciones a través de línea de comandos. Algunas funciones de recuperación no están disponibles en todas las versiones de Windows y la herramienta detecta este tipo de disponibilidad para mitigar posibles errores. p. ej.

Escriba el número de función y presione "Enter" para aplicar estas correcciones.

Antes de empezar:

Se pueden realizar algunos procesos para asegurar el buen funcionamiento del sistema.

1. Reinicie su PC y verifique si el problema persiste.

2. Cree un punto de restauración del sistema.

3. Asegúrese de que la configuración de su fecha, hora y año del sistema sean correctas en su computadora.

4. Deshabilitar el software antivirus.
