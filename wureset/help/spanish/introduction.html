Introducción

La herramienta para restablecer los componentes de Windows Update trabaja a través de la línea de comandos, pero tiene requerimientos básicos como un sistema operativo compatible, ejecutar con los privilegios elevados, etc.

También se requiere que el usuario conozca y acepte los Términos y Condiciones de Uso.

Requerimientos:

Esta herramienta es compatible con Windows 7, Windows 8, Windows 8.1, Windows 10

Cuando se ejecuta esta herramienta se detecta la versión, el nombre y la familia del sistema operativo. p. ej.

Si el sistema operativo no es compatible con esta herramienta, muestra un mensaje de error y la herramienta se cierra. p. ej.

  ir arriba

Permisos:

La herramienta para restablecer los componentes de Windows Update necesita ejeutarse con privilegios elevados. Para ello, haga clic con el botón derecho en él y seleccione "Ejecutar como administrador". De lo contrario, muestra un mensaje de error. p. ej.

  ir arriba

Términos y condiciones de uso:

Debido a que esta herramienta modifica los archivos y la configuración del registro, es importante conocer los términos y condiciones de uso. En cuanto se aceptan, da permiso a la herramienta para realizar estos cambios.

Los Términos y condiciones de uso se muestran en la pantalla en espera de ser aceptados. p. ej.

Presione pulsé "SÍ" y la tecla "Enter" para aceptar. Si no está de acuerdo, puede optar por descontinuar el uso de esta herramienta.
