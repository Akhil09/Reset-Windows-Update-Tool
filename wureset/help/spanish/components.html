Restablecer los componentes de Windows Update

Restablecer los componentes de Windows Update es la primera y principal opción para corregir los problemas en Windows Update.

Sin embargo, este proceso requiere algunos pasos para evitar problemas durante la búsqueda de una solución.

Método adecuado para restaurar componentes:

Antes de comenzar, recuarde hacer una copia de seguridad del registro.

A continuación, seleccione la opción "Restablecer los componentes de Windows Update". Esta opción detendrá los servicios de Windows Update mientras limpia los componentes y restaura los ajustes iniciales. p. ej.

Después, instalar el último agente de Windows Update https://support.microsoft.com/en-us/kb/949104.

Finalmente, reinicie su PC para guardar los cambios.

Si ocurre un error durante el proceso, los servicios de Windows Update pueden encontrarse en mal estado y requieren repararse. p. ej.
