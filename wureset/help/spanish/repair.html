Reparar Windows

Las actualizaciones de Windows y Service Pack pueden fallar al instalar si hay error de corrupción. Por ejemplo, no se puede instalar una actualización si se daña un archivo de sistema.

La herramienta DISM o System Update Readiness puede ayudarle a corregir algunos errores de corrupción de Windows.

DISM para Windows 8, Windows 8.1 y Windows 10:

La administración y mantenimiento de imágenes de implementación (DISM.exe) es una herramienta de línea de comandos que se usa para realizar el mantenimiento de una imagen de Windows.

La herramienta para restablecer los componentes de Windows Update Incorpora líneas sobre el comando DISM para reparar Windows. Las opciones para reparar DISM son:

Escanear la imagen del sistema en busca de errores

Verificar los errores encontradas en la imagen

Reparar los errores en la imagen del sistema

Limpiar los componentes eliminados

Para reparar Windows, seleccione estas opciones en la misma secuencia. Al seleccionar cada opción, debería aparecer un mensaje de proceso. p. ej.

La operación de comando puede tardar varios minutos en completarse. Recuarde reiniciar su PC Después de ejecutar cada comando.

Para Windows 7, use la herramienta System Update Readiness: https://support.microsoft.com/en-us/kb/947821#bookmark-checksur.

Si Windows no puede repararse, debe reinstalar el sistema.
